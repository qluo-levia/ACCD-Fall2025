Dog Walking System – OOP Simulation

Intent

This project explores how simple agents — people, dogs, and environmental obstacles — form a dynamic, emergent system when connected through relationships. By modeling an everyday activity such as walking dogs, the project reveals how individual rules can produce complex group behaviors. User interaction alters the system and generates new patterns, highlighting the idea of emerging properties in a visual and interactive form.

Process

I modeled the system using four primary classes — Person, Dog, Bush, and Environment — to clearly define each element’s behavior. After creating a UML diagram, I implemented the system in p5.js using Object-Oriented programming. The visuals are symbolic, and the user can manipulate people directly, which indirectly affects the dogs through the leash relationship. The system produces emergent behaviors such as clustering, obstacle avoidance, and leash-driven movement patterns.
